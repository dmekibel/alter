## THE NAMED FAILURE PATTERNS (distilled from the KILLED anchors below)
1. **FORCED / DECORATIVE ANALOGY** — a simile/metaphor that sounds relatable but maps loosely or adds no information beyond the plain claim. Every image must map 1:1 AND teach something the plain sentence can't. If in doubt, state it plainly with no image. (This is David's #1 catch. The whole session, the writer reached for images instead of stating the thing.)
2. **WITHHOLDING TEASE** — "I want to show you...", "here's a secret", promising a reveal instead of naming the thing in the same breath. Coy = vague. Name it now.
3. **FLOURISH-ENDING** — a sentence that ends on an ornamental metaphor when the plain version already said it ("the holding you've been living inside", "reached in and changed it").
4. **ESSAY / STIFF CADENCE** — reads written, not spoken. Announces an explanation ("And it is not idle:", "that tension has a cost:"), uses un-contracted formal constructions, list-cadence ("some... some..."). Read every line ALOUD; if a real person wouldn't say it that way, kill it.
5. **MEDITATION-MADNESS** — mystical/weird framing for a zero-context beginner ("thinking you never chose", "barely here", "the observer self").
6. **VAGUE PRONOUN / AUDIENCE-BLIND** — "the noise", "one thing to hold", a referent the reader has to guess; a benefit only a "wellness person" values with nothing for the optimizer.
7. **GENERIC-CATEGORY BEAT** — a phrase any app in the category would use ("your head clears", "calm your body down"), doing no specific work.
8. **SHALLOW** — clear but no real idea; a smart adult learns nothing new about themselves.
9. **CHEESY / MOTIVATIONAL** — "Good, nothing to unlearn", tidy reversals, sweeping "everyone/most people".

## THE ONE DISCIPLINE UNDER ALL OF IT
State the real thing plainly. Reach for an image ONLY when it maps exactly and teaches. Read it aloud. If it sounds like writing, or like a guy trying to sound cool, it dies. Depth comes from a true, non-obvious mechanism said simply, never from a clever phrase.
For DEEP/TEACHING surfaces, plain is not enough and clever is death: the winning move (see the meditation-priming EPIC) is a WARM BUILDING EXPLAINER (School of Life / Kurzgesagt / Barada) that introduces the thing and walks the reader in, so the nuance arrives by teaching, never by a smart line.
