# ALTER — The Genius Gameplan
Convergence of the design brains + [REDESIGN-BRIEF.md](REDESIGN-BRIEF.md). The spine is locked; in-flight brains refine detail, not direction.

## 1. The spine (one paragraph)
A full-screen pixel **guardian who IS you.** One screen, one move. The room is the only scoreboard — no coins, XP, or levels.
Your mirror-self **cannot ship in-world until you ship in real life** — the first system that punishes building-the-system.
It adapts to the hour, your stage, your style and the moment; it **celebrates the deed, never the session** (a 1.5s bloom +
haptic on a real action, then it pushes you off the glass); it turns any goal into a quest, breaks tasks into one-small-step
subtasks, points everything at **getting richer + healthier**, quits your kryptonites the **Allen Carr** way, and **guides — never judges.**

## 2. Load-bearing laws (non-negotiable — from the landed brains)
1. **One screen, one move.** Full-bleed character ~62% of the viewport. No dashboard. (This is the literal fix for "intimidating" + "character too small.")
2. **Send-gate is the north star.** The mirror can't ship until you ship; if ALTER ever proposes building/improving ALTER as the move, it has failed.
3. **No token economy.** Progress is *diegetic* — your room/character become more *you* with each real action. Never coins/XP/streak.
4. **Color celebrates the DEED, never the session.** Calm ~95% of the time; full saturation + haptic only when a real-life action lands; then it resolves and opens the door. Zero juice for opening/scrolling.
5. **Adaptation lives in the engine, never on the surface.** A Context Director composes one face from {time · stage · style · moment}; the screen stays singular. System-building is a signal to **interrupt**, not feed.
6. **Guide, never judge. One small step. Misses are data. Body-first.**

## 3. The converged systems
- **Character = live self-image** (Maltz servo): every action is an identity *vote* that visibly shifts the sprite. Not a cosmetic avatar.
- **Universal goal engine:** any goal → quest (habit / skill-arc / anti-habit). Tasks **decompose into prewritten subtasks** (clean room → bed/table/laundry…). **3-stage skill tracks** (effortful → smoothing → automatic) replace streaks; a quest graduates when it "runs without you."
- **Day loop:** plan (AM) → track live → **graceful re-route when you fall off** (recovery speed is the skill). **Regulation Gate:** withholds the ambitious quest when body data says you're fried (the anti-Heroic move).
- **Adaptation by hour:** morning = generative (extract the one *send*); afternoon = catch the trough; evening = did the send land; night = quiet/subtractive.
- **Outcomes + data:** Get Rich + Get Healthy as the gravity; honest full-life ledger (incl. "waste"); Oura / scale / steps / blood work / finance — manual + Shortcuts now, HealthKit later. **The Gap Meter:** biometrics set the width of your "choose your response" window.
- **Addiction:** Allen Carr reframe (remove the desire, not willpower) + Atomic Habits inversion; "the Loop" rendered as a named antagonist with 3 exits; clean time = growth, never deprivation.
- **Guardian guidance:** guide-not-judge voice, micro-improvement, dopamine-on-improvement, proactive reminders (book the doctor, money check-in), "noticed you went quiet."
- **Engagement / motion / haptics:** 5-tier reward ladder (tap-ack → coin-arc → sprite white-flash → rare gold bloom → redirect line), haptics, meaningful gestures (the Heroic arrow-thunk), time-of-day color washes. *Deep motion pass runs after structure locks.*

## 4. Architecture & phasing (the $0 spine)
Static web (HTML/JS/localStorage) **now** → **Claude-as-brain via iOS Shortcuts** (photo→judge, auto-structure any goal, smart re-plan) → **native iPhone** (Screen-Time block, HealthKit, Core Haptics, camera) → **auto-understanding** (vision judges room/meal). Each phase **earned by use**.
